## Test Automation Framework

This project includes a comprehensive test automation framework. See [README_TESTING.md](README_TESTING.md) for detailed documentation on:
- Running tests
- Writing new tests
- Configuration options
- Reporting and debugging

Quick start:
```bash
# Install dependencies
pip install -r requirements.txt

# Run all tests
pytest

# Run specific test types
pytest -m unit
pytest -m integration
pytest -m smoke
```

# g4j

A Python project with automated test code generation capabilities.

* GT-1 change

## Test Automation Code Generator

This project includes a powerful test automation code generator that can automatically create test cases for your Python code.

### Quick Start

```bash
# Generate pytest tests
python test_generator.py your_module.py

# Generate unittest tests
python test_generator.py your_module.py test_output.py unittest
```

### Features

- ✅ Automatic test generation from Python source code
- ✅ Support for both pytest and unittest frameworks
- ✅ Smart parameter inference based on naming conventions
- ✅ Handles functions and classes
- ✅ Follows Arrange-Act-Assert (AAA) pattern
- ✅ Generates test stubs for edge cases and error handling

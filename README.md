## Test Automation Framework

This project includes a test automation code generator that can create test cases from templates and specifications.

See [test_automation/README.md](test_automation/README.md) for full documentation.

### Quick Start

```bash
cd test_automation
python3 test_generator.py
```

Or generate from JSON specs:

```bash
cd test_automation/examples
python3 generate_from_json.py
```
